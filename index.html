About this Project

This website was developed by Martin Estrin. Martin is a third-year computer science student at the University of Southern California, Viterbi School of Engineering. He is also a Software Engineer specializing in AI Training at Outlier. This semester, Martin is a course producer for CS 170, a first-year computer science class at USC. You can find out more about martin at: Martin's Personal Webiste.

This project uses HTML, CSS, Node.js and a NoSQL database (MongoDB). This project is meant to demonstrate a peer-to-peer tutoring platform where students who have taken AP classes can provide on-demand tutoring for students in need. To see the project in action, first sign up as a tutor in a given subject. Then, visit , AP On-Demand Tutoring, make a user account, and request to be tutored in the same subject. Using a unique algorithm, the program will pair you with a tutor who is qualified to teach your subject and who is currently online and available.
